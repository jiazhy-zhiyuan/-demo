initWithDelegate:pwd:params:delegate:

初始化对象

- (id)initWithDelegate:(NSString *)user pwd:(NSString *)pwd params:(NSString *)params delegate:(id<IFlyDataUploaderDelegate>)delegate

Parameters

user

-[in] 用户名，此用户名为您在语音云论坛中申请的账号,为nil就表明为匿名登陆

pwd

-[in] 密码，密码为您在语音云论坛中申请的账号，当user为空时，pwd自动被置为nil

params

-[in] 登陆的参数，具体参数可以参考用户手册

delegate

-[in] 委托对象

Discussion

如果您在此将params设置为nil,则在上传之前不自动登陆，那么您必须使用IFlySpeechUser自行登陆

uploadData:params:data:

上传数据

- (void)uploadData:(NSString *)name params:(NSString *)params data:(NSString *)data

Parameters

name

-[in] 上传的内容名称，名称最好和你要上传的数据内容相关,不可以为nil

params

-[in] 上传的参数,不可以为nil

data

-[in] 上传的数据，以utf8编码,不可以为nil

Discussion

此函数为数据上传类，上传的过程是异步的，在上传过程中不能将这个对象释放掉
